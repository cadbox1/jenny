# Jenny

A react code generator to create static websites that runs entirely in your browser.

[https://cadbox1.github.io/jenny](https://cadbox1.github.io/jenny)

1.  Write React Components
1.  Generate the React code for them using inputs
1.  Preview the resulting html
1.  Download the index.html
1.  Do it all right in your browser

Named after my mum ❤️
